## Quick Start Examples

<Tabs>
<Tab title="REST API">
```bash
curl -X GET "https://api.byul.ai/api/v2/news?minImportance=8&limit=5" \
  -H "X-API-Key: byul_api_key"
```
</Tab>
<Tab title="JavaScript">
```javascript
const getBreakingNews = async () => {
  const response = await fetch('https://api.byul.ai/api/v2/news?minImportance=8', {
    headers: { 'X-API-Key': process.env.BYUL_API_KEY }
  });
  const data = await response.json();
  console.log(`${data.items.length} breaking news articles found`);
};
```
</Tab>
<Tab title="WebSocket">
```javascript
import { io } from 'socket.io-client';

const socket = io('wss://api.byul.ai/news-v2', {
  auth: { apiKey: 'byul_api_key' }
});

socket.emit('news:subscribe', { minImportance: 7 });
socket.on('news:data', (news) => {
  console.log('Breaking news received:', news.data.news.length);
});
```
</Tab>
</Tabs>
