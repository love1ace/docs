## Getting Started

The Byul AI News API provides access to real-time financial news content from byul.ai. It supports both REST and WebSocket connections for various use cases.

<Info>
Ready to get started? Check out our [Quick Examples](#quick-examples) below or explore the [endpoint documentation](/api-reference/endpoint/get).
</Info>

## Base URL

All REST API requests should be made to:

```
https://api.byul.ai/api/v2
```

## Authentication

All requests require an API key with the `byul_v2_` prefix. Include your API key in the request headers:

```bash
X-API-Key: byul_api_key
```

<Warning>
Keep your API key secure and do not expose it in client-side code. Get your API key from the [Dashboard](https://www.byul.ai/api/dashboard).
</Warning>

## Pricing Plans

View detailed pricing plans and features at [byul.ai/api/pricing](https://www.byul.ai/api/pricing).

**Quick Overview:**
- **Test (Free)**: 100 calls/month, 30 req/min

## Quick Examples

### Fetch Latest News

```bash
curl -X GET "https://api.byul.ai/api/v2/news?limit=10&minImportance=5" \
  -H "X-API-Key: byul_api_key"
```

### JavaScript Example

```javascript
const getNews = async () => {
  const response = await fetch('https://api.byul.ai/api/v2/news?limit=10', {
    headers: {
      'X-API-Key': process.env.BYUL_API_KEY
    }
  });
  
  const data = await response.json();
  console.log('Latest news:', data.items);
};
```

### Python Example

```python
import requests

response = requests.get(
    'https://api.byul.ai/api/v2/news?limit=10',
    headers={'X-API-Key': 'byul_api_key'}
)

news = response.json()
for article in news['items']:
    print(f"{article['title']} - Importance: {article['importanceScore']}/10")
```

## Frequently Asked Questions

**Q: What is the Byul API?**  
The best API service that provides global economic news in real-time without delay. Get important financial market news instantly via REST API and WebSocket.

**Q: What kind of news do you provide?**  
We provide all economic news that affects the market in real-time, including global economic, financial, and corporate news, central bank announcements, and economic indicators. Global news is available in both Korean and English.

**Q: Is there any delay in the news?**  
Even the test plan can be used without delay. Byul aims to provide global economic news alerts as fast as possible without delay. Average response time is under 100ms.

**Q: Is it free to use?**  
A free plan is available for testing purposes. You get 100 API calls per month and 30 requests per minute, with access to real-time economic news.

**Q: What data formats do you support?**  
We provide data in JSON format and support both REST API and WebSocket. All timestamps are in ISO 8601 format (UTC).

**Q: How do I get an API key?**  
You can get an API key by signing up for an account. The issued API key starts with the `byul_v2_` prefix and must be included in all requests as the `X-API-Key` header.
