## Endpoints

### GET /news

Fetch financial news with optional filtering and pagination.

**Parameters:**
- `limit` (optional): Number of items to return (1-100, default: 20)
- `minImportance` (optional): Minimum importance score (1-10, default: 1)
- `symbol` (optional): Filter by stock symbol (e.g., "AAPL")
- `q` (optional): Search query for title/content
- `cursor` (optional): Pagination cursor from previous response
- `category` (optional): Filter by news category

## Rate Limits

Rate limits vary by subscription plan:

- **Test (Free)**: 30 requests/minute
- **Starter ($19/month)**: 60 requests/minute  
- **Pro ($99/month)**: 120 requests/minute
- **Enterprise**: Custom limits

Rate limit information is included in response headers:
- `X-V2-RateLimit-Limit`: Total rate limit
- `X-V2-RateLimit-Remaining`: Remaining requests
- `X-V2-RateLimit-Used`: Used requests
- `X-V2-RateLimit-Reset`: Reset timestamp
