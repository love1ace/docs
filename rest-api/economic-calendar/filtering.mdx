## Path Parameters

The `/economic-calendar/:range` endpoint uses path-based filtering to specify the date range.

### Date Ranges

**`range`** (required)  
Date range for economic events
- **Allowed values**: `today`, `yesterday`, `tomorrow`, `this-week`, `last-week`, `next-week`, `this-month`, `next-month`
- **Example**: `/economic-calendar/today`

<Warning>
Custom date ranges are not supported in V2. Use predefined ranges only.
</Warning>

## Query Parameters

The endpoint supports optional query parameters to customize the response.

### Language Settings

**`lang`** (string, optional)  
Language code for event names and timezone
- **Default**: `en`
- **Available**: `en` (English), `ko` (Korean)
- **Example**: `?lang=ko`
- **Behavior**: 
  - `en`: Returns English event names with GMT-4 timezone
  - `ko`: Returns Korean translations with KST (GMT+9) timezone

## Supported Date Ranges

| Range | Description | Example Use Case |
|-------|-------------|------------------|
| `today` | Today's events | Real-time dashboard, morning briefing |
| `yesterday` | Previous day's events | Analysis, missed events review |
| `tomorrow` | Next day's events | Planning, preparation |
| `this-week` | Current week events | Weekly planning |
| `last-week` | Previous week events | Historical analysis |
| `next-week` | Following week events | Forward planning |
| `this-month` | Current month events | Monthly overview |
| `next-month` | Following month events | Long-term planning |

## Advanced Filtering Patterns

### Multi-Range Calendar View

Fetch multiple date ranges for a comprehensive calendar view:

```javascript
const getCalendarView = async () => {
  const ranges = ['yesterday', 'today', 'tomorrow'];
  
  // Make parallel requests
  const promises = ranges.map(range =>
    fetch(`https://api.byul.app/api/v2/economic-calendar/${range}`, {
      headers: { 'X-API-Key': process.env.BYUL_API_KEY }
    }).then(res => res.json())
  );
  
  const results = await Promise.all(promises);
  
  // Combine results
  const calendarData = ranges.reduce((acc, range, index) => {
    const result = results[index];
    if (result.success) {
      acc[range] = result.data.events;
    }
    return acc;
  }, {});
  
  return calendarData;
};

// Usage
const calendar = await getCalendarView();
console.log('Yesterday:', calendar.yesterday?.length || 0);
console.log('Today:', calendar.today?.length || 0);
console.log('Tomorrow:', calendar.tomorrow?.length || 0);
```

### Currency-Focused Tracking

Monitor events for specific currencies across multiple timeframes:

```javascript
const getCurrencyEvents = async (targetCurrency = 'USD') => {
  const ranges = ['today', 'tomorrow', 'this-week'];
  
  const allEvents = [];
  
  for (const range of ranges) {
    const response = await fetch(`https://api.byul.app/api/v2/economic-calendar/${range}`, {
      headers: { 'X-API-Key': process.env.BYUL_API_KEY }
    });
    
    const data = await response.json();
    if (data.success) {
      const currencyEvents = data.data.events
        .filter(event => event.currency === targetCurrency)
        .map(event => ({ ...event, range }));
      
      allEvents.push(...currencyEvents);
    }
  }
  
  // Sort by datetime
  return allEvents.sort((a, b) => 
    new Date(a.dateTime.utc) - new Date(b.dateTime.utc)
  );
};

// Usage
const usdEvents = await getCurrencyEvents('USD');
const eurEvents = await getCurrencyEvents('EUR');
```

### Importance-Based Filtering

Create different event feeds based on market impact:

```javascript
const createEventFeeds = async (range = 'this-week') => {
  const response = await fetch(`https://api.byul.app/api/v2/economic-calendar/${range}`, {
    headers: { 'X-API-Key': process.env.BYUL_API_KEY }
  });
  
  const data = await response.json();
  if (!data.success) return { high: [], medium: [], low: [] };
  
  const feeds = data.data.events.reduce((acc, event) => {
    acc[event.importance].push(event);
    return acc;
  }, { high: [], medium: [], low: [] });
  
  // Sort each feed by datetime
  Object.keys(feeds).forEach(importance => {
    feeds[importance].sort((a, b) => 
      new Date(a.dateTime.utc) - new Date(b.dateTime.utc)
    );
  });
  
  return feeds;
};
```

### Real-Time Event Monitoring

Set up monitoring for today's events with periodic updates:

```javascript
class EconomicCalendarMonitor {
  constructor(apiKey, language = 'en') {
    this.apiKey = apiKey;
    this.language = language;
    this.lastEventCount = 0;
  }
  
  async checkTodaysEvents() {
    const params = this.language !== 'en' ? `?lang=${this.language}` : '';
    
    const response = await fetch(`https://api.byul.app/api/v2/economic-calendar/today${params}`, {
      headers: { 'X-API-Key': this.apiKey }
    });
    
    const data = await response.json();
    
    if (data.success) {
      const eventCount = data.data.events.length;
      
      if (eventCount !== this.lastEventCount) {
        console.log(`📅 Economic Calendar Update: ${eventCount} events today`);
        
        const highImpactEvents = data.data.events.filter(e => e.importance === 'high');
        if (highImpactEvents.length > 0) {
          console.log(`⚠️  ${highImpactEvents.length} high-impact events today`);
        }
        
        this.lastEventCount = eventCount;
      }
      
      return data.data.events;
    }
    
    return [];
  }
  
  startMonitoring(intervalMs = 300000) { // 5 minutes
    console.log(`🔄 Starting economic calendar monitoring (${intervalMs/1000}s intervals)`);
    return setInterval(() => {
      this.checkTodaysEvents();
    }, intervalMs);
  }
}

// Usage
const monitor = new EconomicCalendarMonitor(process.env.BYUL_API_KEY, 'en');
const monitoringInterval = monitor.startMonitoring();
```

The API automatically adjusts timezones based on the language parameter:

```javascript
const compareTimezones = async () => {
```

```javascript
const getLocalizedEvents = async (language = 'en') => {
```

## Error Handling

### Range Validation

```javascript
const validateRange = (range) => {
  const validRanges = [
    'today', 'yesterday', 'tomorrow',
    'this-week', 'last-week', 'next-week',
    'this-month', 'next-month'
  ];
  
  if (!validRanges.includes(range)) {
    throw new Error(`Invalid range: ${range}. Allowed values: ${validRanges.join(', ')}`);
  }
};

const safeGetEvents = async (range, language = 'en') => {
  try {
    validateRange(range);
    
    const params = language !== 'en' ? `?lang=${language}` : '';
    const response = await fetch(`https://api.byul.app/api/v2/economic-calendar/${range}${params}`, {
      headers: { 'X-API-Key': process.env.BYUL_API_KEY }
    });
    
    const data = await response.json();
    
    if (!data.success) {
      console.error('API Error:', data.error);
      return [];
    }
    
    return data.data.events;
    
  } catch (error) {
    console.error('Request failed:', error.message);
    return [];
  }
};
```

### Comprehensive Error Handling

```javascript
const handleEconomicCalendarErrors = async (range, language = 'en') => {
  try {
    const params = language !== 'en' ? `?lang=${language}` : '';
    const response = await fetch(`https://api.byul.app/api/v2/economic-calendar/${range}${params}`, {
      headers: { 'X-API-Key': process.env.BYUL_API_KEY }
    });
    
    if (!response.ok) {
      throw new Error(`HTTP Error: ${response.status} ${response.statusText}`);
    }
    
    const data = await response.json();
    
    if (!data.success) {
      const { code, message, details } = data.error;
      
      switch (code) {
        case 'INVALID_RANGE':
          console.error('Invalid date range provided:', details);
          break;
        case 'UNAUTHORIZED':
          console.error('API key missing or invalid');
          break;
        case 'INTERNAL_ERROR':
          console.error('Server error occurred');
          break;
        default:
          console.error('Unknown error:', message);
      }
      
      return { success: false, error: data.error };
    }
    
    return { success: true, events: data.data.events, metadata: data.data.metadata };
    
  } catch (error) {
    console.error('Network or parsing error:', error.message);
    return { success: false, error: { message: error.message } };
  }
};
```

## Best Practices

### Efficient Range Selection

1. **Use appropriate ranges**: Choose the most specific range for your use case
2. **Cache results**: Store results to avoid repeated calls for the same range
3. **Consider timezone**: Use the language parameter that matches your user's location
4. **Plan ahead**: Use `tomorrow` and `next-week` for planning purposes

### Performance Optimization

```javascript
// Good: Use specific range
const todayEvents = await fetch('/economic-calendar/today');

// Better: Cache results for repeated access
const eventCache = new Map();

const getCachedEvents = async (range, language = 'en') => {
  const cacheKey = `${range}-${language}`;
  
  if (eventCache.has(cacheKey)) {
    const cached = eventCache.get(cacheKey);
    // Cache for 5 minutes
    if (Date.now() - cached.timestamp < 300000) {
      return cached.data;
    }
  }
  
  const events = await getLocalizedEvents(language);
  eventCache.set(cacheKey, {
    data: events,
    timestamp: Date.now()
  });
  
  return events;
};
```

### Rate Limit Considerations

```javascript
// Avoid: Multiple rapid requests
const badPattern = async () => {
  const today = await fetch('/economic-calendar/today');
  const tomorrow = await fetch('/economic-calendar/tomorrow');
  const thisWeek = await fetch('/economic-calendar/this-week');
  // This may hit rate limits
};

// Good: Batch requests with delays
const goodPattern = async () => {
  const delay = (ms) => new Promise(resolve => setTimeout(resolve, ms));
  
  const today = await fetch('/economic-calendar/today');
  await delay(1000); // 1 second delay
  
  const tomorrow = await fetch('/economic-calendar/tomorrow');
  await delay(1000);
  
  const thisWeek = await fetch('/economic-calendar/this-week');
  
  return { today, tomorrow, thisWeek };
};
```
