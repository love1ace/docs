## Configuration

`windsurf_mcp.json`:

```json
{
  "mcpServers": {
    "mcp-server-byul": {
      "command": "npx",
      "args": ["-y", "@byul/mcp"],
      "env": { "BYUL_API_KEY": "byul_api_key" }
    }
  }
}
```
