---

## Settings file

Open `~/.gemini/settings.json` and add:

```json
{
  "mcpServers": {
    "byul": {
      "command": "npx",
      "args": ["-y", "@byul/mcp"],
      "env": { "BYUL_API_KEY": "byul_api_key" }
    }
  }
}
```

If the `mcpServers` object is missing, create it.

<Tip>
Only local stdio transport is supported. Remote HTTP/SSE endpoints are not available.
</Tip>
